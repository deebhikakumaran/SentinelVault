Follow this design system strictly for all UI work in this project:

## Visual Style: "Ctrl.xyz / Web3 Dark Premium"

### Colors
- Background: #000000 (pure black) or #0A0A0A
- Surface/cards: rgba(255,255,255,0.04) with 1px border rgba(255,255,255,0.08)
- Primary accent: vibrant green (#00FF85 or similar neon green)
- Secondary accent: purple/violet gradient (#8B5CF6 → #6D28D9)
- Text primary: #FFFFFF
- Text secondary: rgba(255,255,255,0.6)
- Subtle borders: rgba(255,255,255,0.1)

### Typography
- Headings: Large, bold, geometric sans-serif (use Inter or Satoshi from Google Fonts). Hero text should be 48-80px, uppercase or sentence case, tight letter-spacing (-0.02em)
- Body: 16-18px, light weight (300-400), generous line-height (1.6)
- Use a period/dot "." as a stylistic element after hero headlines (like "Take Ctrl.")
- Section labels/eyebrow text: small uppercase, letter-spacing 0.1em, muted color

### Layout
- Full-width sections on black background
- Max content width ~1200px, centered
- Generous vertical padding between sections (80-120px)
- Feature cards in 2-3 column grid with rounded corners (16-24px border-radius)
- Cards have subtle glass/frosted effect: semi-transparent bg + thin border

### Components
- Buttons: Rounded-full (pill shape), bright green bg with black text for primary CTA. Ghost/outline buttons with white border for secondary
- Navigation: Minimal top nav, semi-transparent, with pill-shaped CTA button
- Feature cards: Dark surface, icon on top, bold title, short description
- Scrolling marquee/ticker strip for logos, tags, or stickers
- FAQ section: Accordion style with clean dividers

### Effects & Animation
- Subtle hover effects on cards (border glow or slight lift)
- Smooth scroll transitions
- Marquee/infinite horizontal scroll for partner logos or decorative elements
- Gradient glows behind key elements (radial gradient spots of green/purple at ~10% opacity)
- No harsh shadows — use glow effects instead

### Overall Vibe
- Dark, premium, confident
- Minimal but not empty — use accent glows and subtle borders to add depth
- Web3/fintech energy without being cluttered
- Clean information hierarchy with massive hero text stepping down to smaller detail text

Apply this design system to every component, page, and UI element you create.
